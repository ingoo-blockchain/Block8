## JSX

```js
// App 컴포넌트가 있다고 가정합시다.
React.createElement(App, null, "안녕하세요.")

// div Element 에 내용은 안녕하세요 넣어주세요.
React.createElement("div", null, "안녕하세요.")

// babel 의 도움을 받아서 jsx
// React.createElement(App, null, "안녕하세요.") -> JSX
<App>안녕하세요</App>


// React.createElement("div", null, "안녕하세요.")
<div id='asdf'>안녕하세요</div>

<div></div>

<div>Hell World!</div>
```
